The depot setup process

Before you set up a depot, you should:

  • Add countries and regions to the study.
  • Know whether the depot is a central or local depot.
  • Know whether the depot is a drug destruction facility (DDF).
  • Decide on the depot architecture.

During depot setup, you do the following:

  1. Create a new depot.

    For each depot, you identify:

    • Important basic properties—These properties include a unique identifier and name, geographic information, and shipment information.
    • The countries supplied by the depot—The depot can ship to the countries you select. A country can only be supplied by one depot.
    • The local depots supplied by the depot—For a central depot, you must set up local depots that receive drug supplies from this depot. Not all depots are central depots.
    • The depot resupply trigger level—The depot inventory trigger sets the inventory level at which you are alerted that a depot needs to be resupplied.
  2. (optional) Set up the depot as a drug destruction facility (DDF).

    A location can be both a depot and a DDF. The DDF location must be set up as a separate depot with a unique depot ID.

After you set up the depots, you can complete the drug release process.
