Pending Users page

On this page, you edit information for unauthenticated users or retransmit User Authentication Codes (UAC) to users.

Fields on this page

This table describes the default fields for this page. Your study might require different fields, and those fields could have different names.

Field

Description

Notes

Update User Activation Information

 

 

Select User

Selects a pending user.

After you select a user, the Edit User Activation Data form appears.
