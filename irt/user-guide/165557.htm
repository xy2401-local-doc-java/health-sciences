Processing transactions over the phone

After you log in using the phone, the IRT system reads the welcome message and all applicable menu options for your user role. You might have access to one or more of the following transactions on the phone:

Subject Menu

  • Screening
  • Screen Fail
  • Randomization
  • Scheduled Visit
  • Drug Replacement
  • Early Termination
  • Completion
  • Unblind Treatment

Trial Supply Menu

  • Register Site Drug Order Receipt
  • Register Depot Drug Order Receipt
  • Update Site Inventory
  • Update Depot Inventory

To process the above transactions

  • Follow the directions read to you over the phone. Use the following letter codes to enter alpha characters:

  • To return to the main menu, press pound (#).
  • To exit the application, from the main menu, press 0, or hang up.
  • To input special characters:
    • Press star (*) to input a decimal.
    • Enter 27 to insert a dash.

      For example, if a subject does not have a middle initial, the output would be A-B, entered as 012703.
