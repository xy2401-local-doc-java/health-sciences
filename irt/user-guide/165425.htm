Adding a manufacturing lot

You can create a manufacturing lot to:

  • Group finished lots under a manufacturing lots.
  • Track the manufacturing lots in IRT.

Note: The expiration date added to the manufacturing lot is for informational purposes only. IRT does not use these expiration dates to determine the availability of drug units.

  1. Click Supplies ().
  2. In the Set-Up section, select Manufacturing Lots.

    For descriptions of the fields on this page, see Manufacturing Lots page.

  3. Above the list, click Add New Manufacturing Lot.
  4. In the Study Internal ID field, type a unique identifier for the lot.
  5. Select the Expiration Date for the lot.
  6. Type a brief Description of the lot.
  7. Click Submit.
