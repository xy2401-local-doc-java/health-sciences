Editing a supply type

You cannot change the Resupply Number for the supply type.

  1. Click Supplies ().
  2. On the left, in the Inventory section, select Resupply Setting.

    For descriptions of the fields on this page, see Resupply Setting page.

  3. Locate the resupply type.
  4. Click Edit.
  5. Edit the necessary information.
  6. Click Submit.
