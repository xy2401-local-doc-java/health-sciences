Deactivating and reactivating a user

Deactivating and reactivating a user only affects the user's ability to access a specific study. Deactivation does not affect the deactivated user's ability to access other IRT studies. Upon reactivation, the user does not receive a new UAC code. Users can log on to the study with their existing user ID and password.

  1. Click Study ().
  2. On the left, in the Users section, click Edit User.

    For descriptions of the fields on this page, see Edit User page.

  3. From the drop-down lists, select the user by either User ID or Name.
  4. In the Available User Actions section, click Deactivate user or Reactivate user.

    Deactivate or Reactivate appears depending on the user's current access status.
