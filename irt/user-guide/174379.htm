Adding a new label group

Before you set up a label group, you must set up the countries for the study.

  1. Click Supplies ().
  2. On the left, in the Set-up section, select Label Groups.

    For descriptions of the fields on this page, see Label Groups page.

  3. Click Add New Label Group.
  4. In the Name field, type a unique name for the label group.
  5. To identify the countries where drug units can be shipped, click the country in the Countries Not Supplied list, and click Add.

    Use the Ctrl key to select more than one country at a time.

    The countries are added to the Countries Supplied list.

  6. Click Submit.
