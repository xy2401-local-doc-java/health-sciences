Resolving queries in the InForm software and the Clintrial software

In the InForm software, users resolve queries either by:

  • Changing the value of the item that created the query such that the new value passes the rule.
  • Answering the query with the submission of explanatory text.

In the Clintrial software, users resolve discrepancies with the Clintrial Manage or Resolve modules.
