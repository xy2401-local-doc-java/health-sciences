Setting a load-balancing machine online

  1. Select Admin > CIS Sync.

    If only one adapter is configured and registered in the CIS environment, the CIS Synchronization Adapter page appears with the Sync Connections tab selected.

    or

    If multiple adapters are configured and registered in the CIS environment, the Adapter List page appears and displays a list of adapters.

  2. If the Adapter List page appears, click the row of the CIS synchronization adapter whose synchronization connections you want to view.

    The CIS Synchronization Adapter page appears with the Sync Connections tab selected.

  3. Select Load Balancing.

    The Load Balancing tab appears and displays status information about the machines being used for load balancing.

  4. In the Load Balancing table, click Edit next to the load-balancing machine whose status you want to change.

    The Status and Offline Reason fields are enabled.

  5. In the Status column for the machine you want to change, select Online from the drop-down list.
  6. Click Update.

    CIS places the machine online.
