Reviewing an autocoded request

If an administrator configures autocoded requests to require review, you can review coding requests that have been successfully autocoded and have a Needs Review status.

  1. Select Coding Requests.

    The Coding Requests page appears. For a description of the fields on this page, see Coding Requests page.

  2. Select one or more requests with a status of Needs Review ().
  3. Select Actions > Review Complete.

    The request status changes to Coded, awaiting approval () or Completed, awaiting delivery ().
