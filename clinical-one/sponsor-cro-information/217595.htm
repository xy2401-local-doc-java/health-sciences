Verify returned kits to confirm and correct site reconciliation

Learn more about kit reconciliation in the FAQs.

What has to happen first? Site users must reconcile kits that are returned to the site.

To perform source data verification (SDV) on kits returned to the site:

Show me how!

  1. On the Home page, determine where to work:
    • To work with real data in Production mode, click the title of the study.
    • To work with mock data in Testing mode, click the Testing Mode button () on the study.
    • To work with mock data in Training mode, click the Training Mode button () on the study.
  2. Along the top, click Supplies.
  3. Below the study name, click the Inventory tab.
  4. From the Site drop-down, select a site.
  5. Find the correct kits with a combination of the following filters:
    • Select the subject number from the Subject Number drop-down.
    • Find all kits with one or more statuses using the Status drop-down.
    • Narrow the list of kit numbers you want to view with the Number Range fields.
    • In the Search by Kit Number field, enter the kit number and press Enter.
  6. Select a kit that has a gray checkbox in the Verify column.
  7. On the right, below Inventory Management, verify that site reconciliation is correct.
  8. If site reconciliation is correct, click the gray checkbox in the Verify column.

    The checkbox turns green.

  9. If site reconciliation isn't correct:
    1. On the right, in Inventory Management, click Update Kit.
    2. Update the kit as needed, provide a reason, and click Update Kit.

      The checkbox in the Verify column turns green.

    Next steps: You can mark kits that are ready for destruction.

Tips

  • If a site user updates a kit that you verified, the green verified checkbox is removed.
