Mark a kit as missing or damaged at your site

Show me how!

  1. On the Home page, determine where to work:
    • To work with real data in Production mode, click the title of the study.
    • To work with mock data in Training mode, click the Training Mode button () on the study.
  2. Along the top, click Supplies.
  3. Below the study name, click the Inventory tab.
  4. If you have access to multiple sites for the study, select a site from the Site drop-down in the upper-right.
  5. Above the kit list, use the filter to return only the kits you want to mark as missing or damaged:
    1. To view the kits dispensed to a subject, select the subject from the Subject Number drop-down.
    2. To find all kits with one or more statuses, use the Status drop-down.
    3. To narrow the list of kit numbers, use the Number Range.
    4. To search for a specific kit, use the Search by Kit Number field.
  6. Select the kit that is missing or damaged.
  7. On the right, below Inventory Management, click Update Kit.
  8. From the Kit Status drop-down, select the kit's new status, either Missing or Damaged.

    Tip: If a kit experiences a temperature excursion while at your site, select Pre-Quarantined from the Kit Status drop-down.

  9. Enter a reason for the change, and click Update Kit.

Tips

  • If you marked a kit in error as Missing or Damaged, you can set the kit back to Available by following the previous steps.
