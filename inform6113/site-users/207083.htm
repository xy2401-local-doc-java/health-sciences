Enter a comment

Show me how to enter a comment

You can enter a comment on an item, a row of repeating data, or a form.

  1. Click Subjects.
  2. Click the status icon for a subject and visit.
  3. Select a form.
  4. Click a comment icon ( ).
  5. Enter a comment in the Comment box, and click Submit.

    Comment icons and the types of data that they apply to

    What does the comment apply to?

    Which comment icon do I click?

    Nonrepeating data item

    Icon in the last column for the item.

    Item in a row of repeating data

    1. To display individual items, click the number in the first column of the repeating row.
    2. Click the icon in the last column for the item.

    Row of repeating data

    Icon in the last column of the repeating row.

    Form

    Icon in the upper-right corner of the form.
