Scheduling a single report

  1. Run the report.
  2. Save the report.
  3. Click the Schedule icon () for the report.

    The Schedule - [Report Name] page appears.

  4. Specify the following information for the report:
    • Priority.
    • Start and end date.
    • Frequency.
  5. Specify whether you want to override the format, accessibility settings, report language, or delivery settings.
  6. Specify whether you want to override the report prompt default values.
  7. Click OK.
