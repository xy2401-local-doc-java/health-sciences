Migrating to this release of the InForm Publisher software

To migrate an InForm Publisher 2.1.0.1 and later installation to release 6.2:

  1. On the InForm 6.2 server, install InForm Publisher 6.2.
  2. Copy the following files from the old InForm Publisher installation to a location accessible from the InForm 6.2 server:
    • InFormPublisher.config configuration file located in <InForm Publisher Installation Directory>/Bin/Config.
    • nepWslc.txt key file located in <InForm Publisher Installation Directory>/Bin.
  3. On the InForm 6.2 server:
    1. Stop the InForm Publisher Windows service.
    2. Run the utility Migrate.IP.exe located in <InForm Publisher Installation Directory>/Bin.
    3. Start the InForm Publisher Windows service.
