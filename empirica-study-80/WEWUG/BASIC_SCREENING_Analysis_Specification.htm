$$$BASIC$$$SCREENING$$$ Analysis Specification

$$$BASIC$$$SCREENING$$$ is an analysis specification. Like other analysis specifications, you can run the $$$BASIC$$$SCREENING$$$ specification for any or all of the existing dosing category breakdowns and time frames. You must create the $$$BASIC$$$SCREENING$$$ analysis specification before you can:

  • View screening results on the Safety Review tab.
  • View a sector map on the Adverse Events page of the Safety Review tab.
  • Create BLR runs on the Safety Review tab.

An analysis specification includes the Use maximum (instead of most recent) change from baseline option. You can specify a similar option as a safety review configuration option. If you set the safety review configuration option differently than the $$$BASIC$$$SCREENING$$$ specification, a message appears, indicating the different settings.
