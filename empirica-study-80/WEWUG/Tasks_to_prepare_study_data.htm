Tasks to set up study data

To make study data available in Empirica Study, you perform the following tasks:

1.         If the data source is SAS transport files, set up application and study directories on the Empirica Study server.

Note: If the data source is an Oracle database, you set up application and study directories as you register the applications and studies.

2.         Register one or more applications.

3.         Register one or more studies in each application.

4.         Set up a loading and checking run for the studies.

5.         (Optional) Register one or more study pools.

6.         If you registered study pools, set up a loading and checking run for the study pools.
