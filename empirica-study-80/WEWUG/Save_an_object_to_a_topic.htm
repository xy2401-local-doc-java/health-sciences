Save an object to a topic

Before you begin, the application administrator must set up the connection to the Empirica Topics server in the website.properties file. For more information, see the Installation and Upgrade Guide.

1.         Locate or generate the object to save.

For example, to save a report, generate the report.

Note: We recommend that you configure objects before you save them to topics. You cannot make any configuration changes after an object is saved to a topic.

2.         On the page or window where the object appears, click Save to Topic.

Note: The Save to Topic link for a screening analysis result appears in the row menu for the result.

The Save to Topic dialog box appears.

3.         Specify an Attachment Name.

4.         Optionally, specify an Attachment description.

5.         To add the object to an existing topic in Empirica Topics:

A.        Select Add to existing topic.

B.        Type or browse for the Topic name.

When you begin typing, a list of matching topics appears.

C.        Optionally, select an Action name.

6.         To create a new topic in Empirica Topics, and add the object to the new topic:

A.        Select Create New Topic.

B.        Select a Topic template.

C.        Type or browse for the Topic name.

D.        When you begin typing, a list of matching topics appears.

E.        Add the topic to a project:

i.           To add the topic to an existing project, select Add to existing project, and select a project from the drop-down list.

ii.         To add the topic to a new project, select Add to new project named:, and type the name for the new project.

7.         Click OK.
