About subject lists

Empirica Study enables you to create lists of subjects of interest. For example, you can create a list of subjects with a shared demographic profile.

You use subject lists to do the following:

  • View detailed information for each subject in the subject list.

  • Run reports for only the subjects in the subject list.

  • Customize screening analysis types so that results for the analysis type are for only the subjects in the subject list.

  • Generate graphical information for only the subjects in the subject list.

To create subject lists, you can use several methods:

  • On the Subject Lists page, use the query wizard.

  • On the Subject Lists page, create an empty list.

  • On the Subject Lists page, create an XML definition.

  • Throughout the application, drill down from a subject count.
