Using XML to Create a Subject List

When you create and save a query-based subject list, the definition of the subject list is saved in XML. This XML representation can be sent to other users, who can create a new subject list using the XML.

To create a subject list from XML:

1.         Obtain the XML that you want to use. You can copy it from your email if another user sent you the XML via email. (See Emailing a Subject List.)

Note: If you are copying XML from email, copy only the XML (not information about the report). The XML begins below the line XML Definition.

2.         On the Subject Lists page, click Create From XML. The Create Subject List from XML page appears.

3.         In the XML Definition field, paste in the copied XML.

4.         Click Next. The Define Query page appears and is filled in as specified by the copied XML. You can modify the query as needed.
