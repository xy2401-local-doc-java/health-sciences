Creating/Editing a Report Definition

1.         Select an application and study, and go to the Reports tab.

2.         Optionally select a subject list. If you do not select a subject list, then when you run the new report, it will be for all subjects in the study.

3.         On the Report Definitions page, do one of the following:

  1. To create a new report definition, click Create Definition.
  2. To create a Findings Report, click Create Findings Report. If you click Last Findings Report, the settings for the most recent Findings Report that you created from the Report Definitions page are used as defaults; you can change the settings as needed.
  3. To create a Standard Demographics Report, click Create with Standard Demographics.
  4. To create a report definition from XML, click Create Definition from XML.

The Create Definition page appears.

4.         Name the report definition. The Edit Definition page appears.

5.         On the Edit Definition page, specify column attributes.

6.         Optionally edit report attributes.

7.         Optionally edit report descriptors.

8.         Save or run the report definition.
