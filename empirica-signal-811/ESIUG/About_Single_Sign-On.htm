About single sign-on

Single sign-on (SSO) is an authentication method that enables users to log in to the Empirica Signal application and other applications using a single user name and password combination. Users can log in to an SSO environment and access all the applications in the environment without reentering their credentials.

SSO simplifies user administration across applications by enabling you to manage fewer user accounts. Users do not need to remember multiple user names and passwords.

Oracle-hosted installations are configured for SSO with Oracle Health Sciences Identity and Access Management Services. Oracle provides you with a URL to access the IAMS console.

For self-hosted installations, you must do the following to use SSO:

  • Install and configure an SSO application, such as Oracle Access Manager.

  • Install and configure the WebGate agent on the Empirica Signal application server.

Contact Oracle for assistance.

For more information, see Configuring single sign-on in a self-hosted environment.
