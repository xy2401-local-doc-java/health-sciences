Making a validation baseline public

When you create a validation baseline, it is considered temporary. Only you can view and work with it. Subsequent validation jobs that you run overwrite it. To enable other users to view and work with a validation baseline, you must make it public.

Tip: If you don't explicitly make the validation baseline public, creating a deployment package automatically makes it public.

  1. Open the Baselines tab.

    Tip: If you don't see the Baselines tab, at the top left of the page, select View, and make sure Baselines is selected.

  2. Select a baseline and click Make Public.
  3. In the Name field, type the name of the baseline.
  4. In the Description field, type a description for the baseline.
  5. Click OK.
