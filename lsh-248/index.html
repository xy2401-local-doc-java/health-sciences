Oracle Life Sciences Data Hub
Release 2.4.8

Get Started

Oracle Life Sciences Data Hub allows you to build reusable, automated applications to load data from an Oracle database, SAS CPort or XPort file, or text file; merge and transform data using SAS, PL/SQL, or Informatica; write data reports to a hierarchical PDF document; and export data as Oracle Export data marts, SAS CPort or XPort files, or zipped text files. You can view data in visualization tools. Version control and validation tools are built in.

You can merge data from different clinical trials.

Oracle Life Sciences Data Hub also serves as the back end of Oracle Health Sciences Data Management Workbench.
