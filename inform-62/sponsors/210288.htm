Review comments for a single form

  1. Click Review.
  2. Optionally, to sort the page by site, in the upper-left corner of the Data Viewer, click Sites.
  3. To open the Record Listing view:
    1. In the table, select a cell in the Visit, Site, or Country column, or any cell that contains data that you want to see in the Record Listing view.

      For example, if the Summary view is sorted by visit, to open all the forms in the Baseline visit for all sites, select the Baseline cell in the Visit column. To open the Complete forms in the Baseline visit for all sites, select the data cell in the Baseline row and the Complete column.

    2. At the top of the Data Viewer, click Go to Record Listings.
  4. At the top of the Data Viewer, click View Comments ().
    • If text appears in the Item column, the comment is an item-level comment.
    • If text does not appear in the Item column, the comment is a form-level comment.
  5. Optionally, filter and sort the comments, or export the data.
  6. To find the form a comment was entered on, highlight a row and click View in Record Listings.
